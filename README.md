# Suite de Pruebas Automatizadas - Selenium WebDriver

Este proyecto contiene una suite de pruebas funcionales (Caja Negra) para el sitio web **Automation Exercise**, desarrollada como parte del curso de Calidad de Software en la **Universidad Fidélitas**.

## Tecnologías Utilizadas
* **Lenguaje:** Java 17
* **Framework de Testing:** JUnit 5
* **Automatización:** Selenium WebDriver 4.21.0
* **Gestor de Dependencias:** Maven
* **IDE:** Apache NetBeans / IntelliJ

## Cómo Ejecutar el Proyecto

### Pre-requisitos:
1. Tener instalado **Java JDK 17** o superior.
2. Tener instalado **Apache Maven**.
3. Contar con el navegador **Google Chrome** actualizado.

1. Clonar el repositorio:
